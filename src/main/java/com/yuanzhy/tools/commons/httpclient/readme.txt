Commons HttpClient项目过去是Commons的一部分，现在是Apache HttpComponents的一部分。
Apache HttpComponents是Apache的顶级项目，负责创建和维护专注于HTTP和相关协议的Java组件工具集。
所以一般来说看到Apache HttpClient3的说法指的就是Commons HttpClient，包结构是org.apache.commons.httpclient
Apache HttpClient4 指的是Apache HttpComponents下的项目，包结构是org.apache.http
Apache HttpClient5 指的是Apache HttpComponents下的最新项目，包结构是org.apache.hc
HttpClient3早已不在维护，推荐使用最新的HttpClient5，截止本文发布时间2021-08-05，2021-7-12还发布了新版本
